◆ verifyPdfSignatureFromMetadata()

verifyPdfSignatureFromMetadata ( gui,
pdfPath,
publicKeyPath )

Checks whether the digital signature in the PDF document is valid.

The user's RSA (4096-bit) public key is not required during this process.

Parameters
guiThe GUI object used to update the verification status in the user interface.
pdfPathThe path to the PDF document to verify.
publicKeyPathThe path to the file that stores the user's public key.
Returns
Returns OK if the process was successful.
Returns NO_SIGNATURE if the PDF document does not contain a digital signature.
Returns SIGNATURE_VERIFICATION_VALUE_ERROR if the digital signature is not valid.
Returns UNKNOWN_ERROR if an exception was encountered.
