Searches for a file containing the user's private key on mounted pendrives and reads its contents.

Parameters
pendrivesList of mounted pendrives.
filePathRelative path to the file containing the user's private key.
Returns
Returns the private key as bytes if the file was found and successfully read. On the other hand, returns the string "Not found" if no file has been found.
